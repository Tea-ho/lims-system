[[overview]]
= Overview

LIMS (Laboratory Information Management System) Study 프로젝트의 REST API 문서입니다.

[[overview-http-verbs]]
== HTTP verbs

|===
| Verb | Usage

| `GET`
| 리소스 조회에 사용

| `POST`
| 새로운 리소스 생성에 사용

| `PUT`
| 기존 리소스 수정에 사용

| `DELETE`
| 기존 리소스 삭제에 사용
|===

[[overview-http-status-codes]]
== HTTP status codes

|===
| Status code | Usage

| `200 OK`
| 요청이 성공적으로 완료됨

| `201 Created`
| 새로운 리소스가 성공적으로 생성됨

| `204 No Content`
| 요청이 성공했으나 응답 본문이 없음

| `400 Bad Request`
| 잘못된 요청 형식

| `401 Unauthorized`
| 인증이 필요함

| `403 Forbidden`
| 권한이 없음

| `404 Not Found`
| 요청한 리소스를 찾을 수 없음
|===

[[resources-user]]
== User API

사용자 관리 API입니다.

=== 사용자 생성

`POST /api/users`

==== Request Fields

|===
| Path | Type | Description | Constraints

| `username`
| String
| 사용자 아이디
| 필수, 3-50자

| `password`
| String
| 비밀번호
| 필수, 8-100자, 대소문자+숫자+특수문자 포함

| `role`
| String
| 권한
| 필수, ROLE_USER/ROLE_MANAGER/ROLE_ADMIN
|===

==== Response Fields

|===
| Path | Type | Description

| `success`
| Boolean
| 성공 여부

| `data.id`
| Long
| 사용자 ID

| `data.username`
| String
| 사용자 아이디

| `data.role`
| String
| 권한

| `data.createdAt`
| String
| 생성일시

| `data.updatedAt`
| String
| 수정일시
|===

==== Example Request

```
POST /api/users HTTP/1.1
Content-Type: application/json

{
  "username": "testuser",
  "password": "Password123!",
  "role": "ROLE_USER"
}
```

==== Example Response

```
HTTP/1.1 201 Created
Content-Type: application/json

{
  "success": true,
  "data": {
    "id": 1,
    "username": "testuser",
    "role": "ROLE_USER",
    "createdAt": "2025-01-01T00:00:00",
    "updatedAt": "2025-01-01T00:00:00"
  }
}
```
